# Python Lab Verifier

This project provides a Dockerized environment to automatically verify Python lab assignments based on specific student variants.

## Prerequisites For Each Submission

* **Docker** installed on your system (Ubuntu, macOS, or Windows).
* Student code entrypoint named `main.py`.
* Setup script named `set_variant.sh`.
* Verification script named `verify.sh`.

## Getting started

Each submission has own *README.md* in which is set verification script
